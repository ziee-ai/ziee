# TESTS — split-chat-multipane

Frontend-only feature → **unit** (`*.test.ts` colocated, run under vitest/tsc)
and **e2e** (Playwright) tiers only; no backend integration tier (no server
diff). New e2e specs live under `src-app/ui/tests/e2e/14-split-chat/`
(`--workers=1`). Every ITEM is covered; the user-visible flows each get an e2e
spec. Mock only the external boundary (the SSE transport / API cassette), never
the behavior being asserted.
